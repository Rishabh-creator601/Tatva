# Objective 
Caraousel is a para-diagram of the tatva panel. 


# naming 
carousel item /item = can be a image/note/text content  or even a  link  



# Carousel  properites and Nature 

Every carousel that is Formed will be a scrollable element 
each carousel will have own colour and can be changed 
A separate message can be added to the carousel header as well, despite the individual item. 
When watching the carousel, the background got blurred and focus on the current item will be done. 
It may contain notes, that is, text material or images. 
Every item of the carousel will have a delete option, moving to other directory/item, editing it, Adding a special message to it 
Items that are marked as important will be shown first on the carousel. 
The carousel itself's hold can be marked as important, hence shown first on the menu item page  , with a different colour 
The carousel, can be marked for the home page seperately . With a different color 
The important thing here is that the carousel shown on the top of the page and the carousel shown on the home page are to be marked separately. 
The carousel can have a mix of images and notes as well. 
Carousel items can be imported from a folder, or multiple items can be imported by selecting them. 
If it is marked as important, then it will be displayed on the home page until it is unmarked manually. 
The whole carousel can be exported And will be saved on the desktop 
Carousel item  File name can be changed 
Copy to clipboard functionality also available 
Every carousel that will be formed will have a number assigned
You can rearrange the carousels and assign it a new item number. And it will be as it is until rearranged manually. 
one can edit the caraousel name itself and delete the carousel as well 

the caraousel properties will be shown by a settings icon besides the title with the name of functionality except the rearranging one
